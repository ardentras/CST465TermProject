  • Term Project Idea!

  • The World's Least Secure Password Manager

  • Or else a simple note/information storage app

  • This application will contain user logins and allow users to store information based on either certain pre-formatted forms (such as server connect information, website login credentials, etc) or else a general note interface with the option to create certain custom fields. This information will be stored in a database and retrievable following a user logging into their account. This application will provide no security outside of not using the built-in HTML authentication since this is not really a security focused course.
